## 比較分析手順

### 前提
- 情報収集フェーズが完了していること
- 比較軸が定義されていること

### 手順
1. 比較軸ごとに分析を実施する
2. 比較表を作成する（Markdown表形式、全対象 x 全軸のマトリクス）
3. 各セルに根拠（データ出典）を付記する
4. 不確実性・留意点を洗い出して記載する
5. 結論と推奨を比較結果から導出する

### 必須セクション
- **比較軸の定義と選定理由**（なぜこの軸で比較するのか）
- **比較表**（全対象 x 全軸のマトリクス）
- **不確実性・留意点**（データの限界、前提条件）

### 品質チェック
- [ ] 比較軸が明確に定義され、選定理由が説明されている
- [ ] 比較表が全対象 x 全軸を網羅している
- [ ] 不確実性セクションがある
- [ ] 結論が比較結果と整合している
